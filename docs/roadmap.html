Roadmap

Curious about what we're building next?

Last updated: July 2, 2025

Please take the dates with a grain of salt, I'm actively making Denjin better, but things may take longer than expected.

Completed

  • Support local models (shipped in v0.2.0)
  • Supabase support (first shipped in v0.2.0, graduated in v0.3.0)
  • Linux support (beta support in v0.3.0)
  • Upload image (shipped in v0.4.0)
  • Use any AI model (shipped in v0.5.0)
  • Import existing projects (shipped as experimental feature in v0.6.0)
  • Support larger codebases (shipped as Smart Context pro mode in v0.6.0)
  • Next.js support (shipped in v0.7.0)
  • Copy app (shipped in v0.7.5)
  • Select UI to edit (shipped in v0.8.0)
  • Ask mode (shipped in v0.9.0)
  • Create mobile app with Capacitor (shipped in v0.10.0)
  • Auto-fix errors (shipped in v0.11.0)
  • Support other tech stacks besides React - anyone can now add a template!
  • Undo-able database (shipped in v0.16.0 Beta 1)

Short-term

August 2025

  • Graduate Portal template from experimental
  • Create a hub/marketplace to share prompts, templates, and plugins
  • Run Denjin in Docker container

Medium-term

  • Prototype MCP support
  • Planning support
  • Support importing from Figma
